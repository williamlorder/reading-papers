Supplementary Materials:
Domain-Specific Gender Gaps in Sexual Attitudes

[Blinded for Review]

S1. Variable Coding and Measurement

S1.1 Dependent Variables

Sexual attitude items are drawn from the CGSS morality module, which asks respondents to evaluate specific sexual behaviors on a scale from 1 (“always wrong”) to 5 (“not wrong at all”):

  • Premarital sex: “What is your opinion about sexual relations before marriage?” (a38)

  • Extramarital sex: “What is your opinion about a married person having sexual relations with someone other than his or her spouse?” (a39)

  • Same-sex relations: “What is your opinion about sexual relations between two adults of the same sex?” (a40)

Higher values indicate more accepting attitudes.

S1.2 Gender Ideology Index

The gender ideology index is constructed from five items (a421–a425), each rated on a 1–5 agree-disagree scale:

  1. “Men should focus on career, women should focus on family” (a421)

  2. “Men are inherently more capable than women” (a422)

  3. “For women, marrying well is more important than doing well at work” (a423)

  4. “When jobs are scarce, women should be fired first” (a424)

  5. “Housework is primarily women’s responsibility” (a425)

The index is the mean of all available items (Cronbach’s \alpha = 0.72). Higher values indicate more patriarchal gender ideology. For regression models, the index is standardized (z-scored).

S1.3 Control Variables

  • Age: Continuous

  • Age^2/100: Quadratic age term divided by 100 for scaling

  • Education: Years of formal schooling (continuous)

  • Married: Binary (1 = currently married or cohabiting)

  • CCP membership: Binary (1 = member of the Chinese Communist Party)

  • Religious: Binary (1 = identifies with any religion)

  • Internet user: Binary (1 = uses the Internet)

  • Urban: Binary (1 = urban residence)

  • Agricultural hukou: Binary (1 = agricultural household registration)

  • Ln(income): Natural log of annual personal income

  • Survey year: Fixed effects (factor)

S1.4 Missing Data

Missing values were imputed using the missRanger algorithm with predictive mean matching (k = 5). This method combines random forest imputation with nearest-neighbor matching to preserve the distributional properties of the original data. All variables in the analytic dataset were included in the imputation model.

S5. Robustness and Sensitivity Analyses

This section consolidates all robustness checks and sensitivity analyses. We assess the stability of the main findings across alternative model specifications (ordered logistic regression, province fixed effects), alternative estimation strategies (propensity score matching, inverse probability weighting, listwise deletion), supplementary tests (gender ideology interaction, wave-specific estimates), and formal sensitivity analyses (causal mediation sensitivity, E-values).

S5.1 Ordered Logistic Regression

To verify that results are robust to the ordinal nature of the dependent variables, we re-estimated all main models using ordered logistic regression (proportional odds model).

S5.2 Province Fixed Effects

To address concerns about unobserved provincial heterogeneity, we re-estimated main models with province fixed effects.

S5.3 Gender Ideology \times Gender Interaction

To formally test whether gender ideology operates differently for men and women, we estimated models with a female \times gender ideology interaction term.

S5.4 Wave-Specific Estimates

To assess the stability of key coefficients across survey waves, we estimated the main model specification separately for each CGSS wave (2012–2023). Figure 3 plots the female coefficient and gender ideology coefficient by wave.

S5.5 Causal Mediation Sensitivity Analysis

Following Imai et al. (2010), we conducted sensitivity analysis for the causal mediation estimates using the medsens() function from the mediation R package. The sensitivity parameter \tilde{\rho} represents the correlation between the residuals of the mediator and outcome models. We report the value of \tilde{\rho} at which the ACME equals zero: \tilde{\rho} = -0.010 for premarital sex, \tilde{\rho} = 0.050 for extramarital sex, and \tilde{\rho} = -0.050 for same-sex relations. These values are small, indicating that the mediation estimates are sensitive to unmeasured confounding: a confounder inducing a residual correlation of approximately 0.05 would suffice to nullify the extramarital and same-sex mediation effects.

S5.6 E-Value Sensitivity Analysis

Following VanderWeele and Ding (2017), we calculated E-values for the female coefficients in each domain. The E-value for the premarital sex coefficient is 1.48 (CI limit: 1.45), for extramarital sex 1.56 (CI limit: 1.52), and for same-sex relations 1.39 (CI limit: 1.35). These moderate thresholds suggest that unmeasured confounding alone is unlikely to fully explain the observed gender gaps, though they indicate that the same-sex domain is somewhat more vulnerable to confounding explanations.

S5.7 Summary of All Robustness Specifications

Table S7 summarizes the female coefficient across all robustness specifications. The direction, magnitude, and statistical significance of the gender gap in each domain are consistent across ordered logistic regression, province fixed effects, propensity score matching (1:1 nearest-neighbor with caliper 0.2, N = 65,350), inverse probability weighting (stabilized, trimmed at 99th percentile), and listwise deletion without imputation. E-values and sensitivity parameters from mediation analysis are also reported. We note that because gender is not a manipulable treatment, the PSM and IPW analyses should be interpreted as sensitivity checks for covariate imbalance rather than as causal treatment effect estimates.
